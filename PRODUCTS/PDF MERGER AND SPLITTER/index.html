PDF MERGER AND SPLITTER

Merge multiple PDF files into one or split large PDFs into individual pages or sections. Supports batch processing, allowing you to handle multiple files at once for maximum efficiency.

Features

  • Add Files: Select multiple PDF files to merge or split.
  • Delete File: Remove selected files from the list before processing.
  • Clear List: Remove all added files at once.
  • Merge PDFs: Combine multiple PDFs into a single document.
  • Split PDFs: Separate a PDF into individual pages or specific sections.
  • Save Output: Choose the destination folder for the processed files.
  • Batch Processing: Process multiple PDFs at once for efficiency.

System Requirements

  • Operating System: Windows XP/Vista/7/8/8.1/10.
  • Processor: Pentium 4 or higher processor.
  • Ram 1GB.
  • Harddrive Space: 500 MB
